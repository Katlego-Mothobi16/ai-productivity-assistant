# AI Workplace Productivity Assistant

## Project Overview

AI Workplace Productivity Assistant is a modern, responsive web application designed to help professionals automate everyday workplace tasks using Artificial Intelligence. The application provides AI-powered tools for generating professional emails, creating structured task schedules, and interacting with an AI workplace assistant—all within a clean, SaaS-inspired interface.

The application is built as a client-side web application with **no user authentication, no database, and no persistent storage**, making it simple, lightweight, and easy to deploy.

---

## Features Implemented

### 📧 Smart Email Generator
- Generate professional emails using AI.
- Support for multiple writing tones:
  - Formal
  - Friendly
  - Persuasive
- Editable AI-generated email output.
- Copy and regenerate functionality.
- Uses structured AI prompts for consistent results.

### 📅 AI Task Planner
- Generate daily or weekly work schedules.
- Prioritise tasks based on urgency and importance.
- Suggest estimated time allocations.
- Provide productivity recommendations.
- Editable and regenerable task plans.

### 💬 AI Chatbot Interface
- Interactive AI workplace assistant.
- Responds to workplace-related prompts.
- Assists with communication, planning, and productivity tasks.
- Maintains conversation only during the current browser session.

### 🎨 Modern Dashboard
- Responsive SaaS-style interface.
- Sidebar navigation.
- Mobile-friendly design.
- Clean and intuitive user experience.

### ⚠️ Responsible AI
- Includes a Responsible AI disclaimer.
- Encourages users to review AI-generated content before use.
- Advises against entering confidential or sensitive information.

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/ai-workplace-productivity-assistant.git
```

### 2. Navigate to the Project Folder

```bash
cd ai-workplace-productivity-assistant
```

### 3. Install Dependencies (if applicable)

```bash
npm install
```

### 4. Configure the AI API

Add your AI API key according to your chosen provider's documentation (for example, using an environment variable or configuration file).

### 5. Start the Application

If using a development server:

```bash
npm run dev
```

Or simply open the project in your browser if it is built as a static web application.
